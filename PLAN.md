# Нормоконтроль ВКР — Автоматическое исправление

**Проект:** Расширение [normokontrol-vkr-astafev](https://github.com/astafiev-rustam/normokontrol-vkr-astafev)  
**Репо:** https://github.com/10q4attention/normokontrol-autofix  
**Цель:** Добавить к проверке автоматическое исправление оформления `.docx`  
**Стек:** Python 3.10+, Flask, python-docx  
**Статус:** Фазы 1 и 2 завершены → переходим к Фазе 3 (UI)

---

## Идея

Программа препода только **проверяет** и выдаёт отчёт об ошибках. Мы добавляем режим **«Проверить и исправить»**: система находит нарушения и сама правит `.docx`, сохраняя оригинал как резервную копию.

## API

### POST /api/fix
```json
// Вариант 1 — конкретный файл
{ "file_path": "/path/to/diploma.docx" }

// Вариант 2 — папка со студентами (как в /api/scan)
{ "folder_path": "/path/to/students/" }
```

Ответ:
```json
[{
  "status": "ok",
  "file_path": "...",
  "backup_path": "..._backup_20250515_143022.docx",
  "total_changes": 5,
  "results": [
    { "fixer_id": "page_layout", "name": "Макет страницы",
      "status": "ok", "changes": ["Секция 1: левое поле 25.4→30 мм"] },
    { "fixer_id": "main_text", "name": "Основной текст",
      "status": "ok", "changes": ["Исправлено абзацев: 47"] }
  ]
}]
```
